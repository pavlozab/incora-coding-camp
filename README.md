## Installation

### Required to install

- NodeJS (^14.0.0)

### Clone

- Clone this repo to your local machine using `https://github.com/pavlozab/incora-coding-camp`

### Setup

> install npm packages

```shell
$ npm install
```

### How to run

>

```shell
# unit tests
$ npm test

# start
$ npm start

# development
$ npm run start:dev
```
