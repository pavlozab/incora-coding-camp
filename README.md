## Installation

### Required to install

- NodeJS (^14.0.0)

### Clone

- Clone this repo to your local machine using `https://github.com/pavlozab/incora-coding-camp`

### Setup

> install npm packages

```shell
$ npm install
```

### How to run

> tests

```shell
$ npm test
```

> start

```shell
$ npm start
```

> development

```shell
$ npm run start:dev
```

> production mode

```shell
$ npm run start:prod
```
